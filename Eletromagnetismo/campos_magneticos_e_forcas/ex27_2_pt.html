Exploração 27.2 - Seletor de velocidade

 

Por favor espera até a animação estar completamente carregada.

Descrição da Animação

Um espectrómetro de massa permite medir a massa das partículas. O primeiro passo no seu funcionamento é selecionar partículas com uma determinada velocidade. À medida que exploras esta animação vais perceber como é que funciona este seletor de velocidade. A animação mostra uma partícula com carga positiva, que atravessa uma região com um campo magnético constante que aponta para dentro do ecrã.  Início

 

Exploração

  1. ANTES de correr a animação, PREVÊ o percurso que a carga vai fazer. Já fiz a minha previsão, mostrem-me o percurso correto.  A tua previsão estava correta? Se não, qual foi o erro?

Agora supõe que um campo elétrico constante é adicionado à região onde já existe um campo magnético. 

  1. Em que sentido (direita, esquerda, para cima, para baixo, para dentro ou para fora do ecrã) deve ser orientado o campo elétrico, de forma a que seja possível equilibrar a força devida ao campo magnético? 

  2. Para criar o campo elétrico são usadas duas placas carregadas. Qual das placas deve ter carga negativa e qual deve ter carga positiva, para criar o campo elétrico desejado?

  3. Já fiz a minha previsão; quero verificar a minha resposta.  A tua previsão estava correta? Se não, o que causou o teu erro?

  4. Deduz uma relação matemática entre o campo elétrico, o campo magnético e a velocidade que a partícula deve ter para passar pela região, sem que a sua trajetória seja afetada (isto é, sem que seja defletida).  

  5. Na animação, o campo elétrico produzido pelas placas é 6000 N/C e o campo magnético é 0.3 T.  Usa a relação da alínea anterior para calculares a velocidade que a partícula deve ter para atravessar a região em linha reta. Usa a caixa em baixo para verificares se o teu resultado está correto.    

Velocidade inicial = x 100 m/s
