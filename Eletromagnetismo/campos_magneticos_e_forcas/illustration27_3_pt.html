Ilustração 27.3 - Espectrómetro de Massa

Demonstração com várias massas   Para uma partícula só:  

velocidade inicial = m/s          campo elétrico  = N/C           campo magnético  = T  

massa  = x 10-3 kg           carga  = x 10-3 C

Por favor espera até a animação estar completamente carregada.

Descrição da Animação

Nesta animação, partículas com carga e com massa atravessam uma zona onde é aplicado um campo elétrico vertical para baixo e um campo magnético horizontal que aponta para dentro do ecrã. Após atravessarem esta zona entram noutra onde existe apenas campo magnético. Início

 

Sabias que...

Observa a Demonstração com várias massas. As partículas têm massas diferentes, mas são idênticas em tudo o resto. Nota que as partículas são separadas com base na sua massa. Este é o princípio de funcionamento de um espectrómetro de massa!
