Descrição da Animação

Nestas colisões não existem forças externas a atuar nos carrinhos. É possível escolher valores para a velocidade inicial de cada carrinho e para a massa do carrinho que se desloca para a direita (amarelo escuro).  Carrega em "registar valores e começar" para ver a animação (a posição é dada em metros e o tempo em segundos). Foram estabelecidos limites para os valores que podes escolher:

0,25 kg < m1 < 4,00 kg,             0 m/s < v1  < 4 m/s,            e        -4 m/s <  v2  < 0 m/s.

Na tabela está indicado o valor do momento linear de cada carrinho, assim como o momento linear total do sistema dos dois carrinhos. Podes ainda selecionar a caixa que permite ver as setas que representam os valores das velocidades relativas, antes e depois das colisões entre os dois carrinhos. Início
